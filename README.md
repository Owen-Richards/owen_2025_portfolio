### Prerequisites
- Node.js 18+

### Installation

```bash
# Clone the repository
git clone https://github.com/owen/portfolio-2025.git
cd owen_2025_portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash

npm run build

# Start production server

```
